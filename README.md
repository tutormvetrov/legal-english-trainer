# Legal English Trainer

Десктопное приложение для изучения 1000 юридических терминов (английский–русский) с использованием интервального повторения (алгоритм SM-2). Работает полностью офлайн, доступно для Windows и macOS.

---

## Установка

### Windows
1. Скачай `LegalEnglishTrainer.exe` из [GitHub Releases](https://github.com/tutormvetrov/legal-english-trainer/releases/latest)
2. Запусти файл напрямую — установка не нужна

### macOS
1. Скачай `LegalEnglishTrainer.dmg`
2. Открой DMG, перетащи приложение в папку Applications
3. При первом запуске: правый клик → **Открыть** (обход Gatekeeper)

### Запуск из исходников
```bash
pip install -r requirements.txt
python run.py
```

## Активация

При первом запуске приложение попросит ввести серийный ключ (формат `XXXXX-XXXXX-XXXXX-XXXXX`) и твоё имя. После активации ключ сохраняется локально — повторная активация не нужна.

## Режимы обучения

### 🃏 Карточки
Основной режим с алгоритмом SM-2. Термин показывается на карточке, ты оцениваешь своё знание:
- **Показать перевод** — открывает перевод, определение и пример использования
- **Знаю** — термин уходит в очередь на длительный интервал
- **Не знаю** — термин вернётся завтра

Каждые N карточек (настраивается) появляется мини-тест. Можно выбрать категорию и направление перевода (EN→RU или RU→EN). Кнопка 🔊 озвучивает термин. Кнопка ★ добавляет в избранное.

### 🔗 Сопоставление
Игра: два столбца — термины слева, переводы справа (перемешаны). Кликни термин, затем его перевод. При верном совпадении — зелёная подсветка, при ошибке — красная с анимацией тряски.

Настройки: категория и количество пар (3–10).

### ✍️ Диктант
Видишь термин — вводи перевод с клавиатуры. Есть 3 попытки: чем меньше попыток потрачено, тем выше оценка для SM-2. Поддерживается направление EN→RU и RU→EN.

### 🔍 Детектив
Тебе показывают **определение** термина. Сам термин скрыт — только первые буквы каждого слова:

```
A _ _ _ _ _ _ _ _ _  a _ _ _ _ _ _ _ _ _
```

- **Открыть букву** — раскрывает одну случайную букву (−10 очков)
- Введи термин целиком в поле и нажми **Проверить** (неверный ответ −25 очков)
- **Сдаться** — показывает ответ, SM-2 получает минимальную оценку

Начальный счёт — 100 очков. Чем меньше подсказок — тем выше итоговый балл.

### 📖 Контекст
Показывается пример предложения с пропущенным термином (___). Введи правильное слово. Режим работает только с терминами, у которых есть пример использования.

### ⭐ Избранное
Список всех терминов, помеченных звёздочкой. Есть поиск по EN и RU тексту, панель с полным описанием выбранного термина, кнопка 🔊 для озвучки.

### 👊 Boss Mode
Режим на скорость и реакцию:
- Показывается термин на английском
- 4 варианта перевода, один правильный
- Таймер убывает — нужно успеть ответить
- Каждый верный ответ сокращает время на следующий вопрос
- Ошибка или истекшее время → конец игры
- Рекорд сохраняется между сессиями

### 📊 Статистика
- Сводные показатели: всего терминов, изучено, прогресс %, повторений сегодня, средний коэффициент лёгкости
- Таблица прогресса по категориям
- Список 20 самых сложных терминов (с наибольшим числом ошибок)
- Кнопки: экспорт в CSV, импорт терминов, резервная копия/восстановление БД, сброс прогресса

## Термин атакует ⚔

Режим фонового обучения. Включается в **Настройках → Режим атаки**.

Раз в N минут поверх всех окон появляется небольшой попап с термином. Закрыть его нельзя — нужно либо правильно ввести перевод, либо нажать «Пропустить» (засчитывается как ошибка в SM-2).

После правильного ответа попап закрывается автоматически через 1 секунду.

## Настройки

Открыть: кнопка **⚙ Настройки** в нижней строке окна.

| Параметр | Описание | Дефолт |
|----------|----------|--------|
| Размер шрифта | Глобальный размер UI (10–20 px) | 13 px |
| Цель на день | Сколько карточек нужно повторить в день | 20 |
| Квиз каждые | Через сколько карточек запускается мини-тест | 10 |
| Начальное время Boss Mode | Стартовый таймер в мс | 5000 мс |
| Ежедневное напоминание | Уведомление в трее в заданное время | Выкл |
| Режим атаки | Включить «Термин атакует» | Выкл |
| Интервал атаки | Как часто появляется попап | 30 мин |

## Стрик и ежедневная цель

В заголовке окна отображается текущий стрик — количество дней подряд, когда ты открывал приложение (🔥 N дней подряд при 2+ днях). Стрик сбрасывается, если пропустить день.

В строке состояния внизу отображается прогресс дневной цели: `Цель на день: X / Y` с полоской прогресса.

## Импорт своих терминов

Открой **Статистика → Импорт терминов**.

**CSV формат** (с заголовком):
```
term_eng,term_rus,definition,category,example
Offer,Оферта,A proposal to enter into a contract,Contract Law,The offer was accepted in writing.
```

**JSON формат**:
```json
[
  {
    "term_eng": "Offer",
    "term_rus": "Оферта",
    "definition": "A proposal to enter into a contract",
    "category": "Contract Law",
    "example": "The offer was accepted in writing."
  }
]
```

Обязательные поля: `term_eng` и `term_rus`. Дубликаты (по `term_eng`) пропускаются.

**Режим вставки**: скопируй текст в формате `термин[TAB]перевод`, вставь в поле — разделители настраиваются.

## Управление данными

В разделе **Статистика** доступны:

- **Экспорт в CSV** — сохранить прогресс в таблицу (для анализа в Excel)
- **Резервная копия** — сохранить файл базы данных в выбранное место
- **Восстановить** — загрузить ранее сохранённую резервную копию
- **Сбросить** — два варианта:
  - *Сбросить прогресс* — удалить историю повторений (термины и избранное сохраняются)
  - *Заводские настройки* — сбросить прогресс + стрик, сменить имя пользователя (ключ активации сохраняется)
